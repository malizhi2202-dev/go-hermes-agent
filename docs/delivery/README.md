## 文档列表

- `go-complete-architecture-and-optimization.md`
  - Go 版本完整架构、方案、取舍理由、论文启发和后续优化路线
  - 适合项目把控、架构评审、开发接手、阶段汇报

- `go-complete-architecture-and-optimization.docx`

## 建议使用方式

1. 先看 `go-complete-architecture-and-optimization.md/.docx`

   - 理解各部分为什么采用当前方案

2. 再按需要查阅支撑文档
   - `../architecture/`
   - `../analysis/`
   - `../migration/`
