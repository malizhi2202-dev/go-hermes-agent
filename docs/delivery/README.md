# Go 交付文档包

这个目录提供一套面向学习、管理和迁移把控的交付文档。

## 文档列表

- `go-learning-and-control-pack.md`
  - Go 版本全方位学习与把控文档
  - 面向老板、架构负责人、项目负责人、接手开发者

- `go-learning-and-control-pack.docx`
  - 上面文档的 Word 版本

- `go-mr-execution-plan.md`
  - 任务拆解与 MR 文档
  - 面向阶段汇报、任务推进、MR/验收管理

- `go-mr-execution-plan.docx`
  - 上面文档的 Word 版本

## 建议使用方式

1. 先看 `go-learning-and-control-pack.md/.docx`
   - 建立整体认识

2. 再看 `go-mr-execution-plan.md/.docx`
   - 明确推进顺序和验收方式

3. 配合已有文档一起读
   - `../architecture/`
   - `../analysis/`
   - `../migration/`
   - `../security/`
